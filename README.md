# Credit Wise Loan System

This project implements an Intelligent Loan Approval System using Machine Learning in Python.
The system predicts whether a loan application should be Approved or Rejected based on applicant details such as income, credit history, and employment status.

# WORKFLOW ---
* Data Visualization: Used plots (pieplot, barplot, boxplot, heatmap) to understand data patterns
* Data Encoding: Converted categorical data into numerical using Label Encoder and One Hot Encoder
* Feature Engineering: Handled missing values, created new features, removed unnecessary columns
* Training & Testing: Split dataset into training and testing sets
* Models Used: Logistic Regression, KNN, Naive Bayes
* Evaluation: Compared models using accuracy score — NAIVE BAYES performed BEST with 84.5% accuracy
